# Glossary

- `Orchestrator PM`: The always-on hub PM skill that routes work and enforces gates.
- `Team PM`: Optional PM for a specific team/domain in multi-team mode.
- `Role Skill`: On-demand specialist skill selected by task match.
- `Signal`: External idea, repo, post, or article pending triage.
- `Registry`: Canonical definitions used by PM routing and governance.
- `Tool Stack Profile`: Predefined install-ready tool/MCP bundle selected by project type.
- `Token Cost Protocol`: Mandatory operating defaults that minimize unnecessary model/tool/context token burn.
- `Forward Scenario Scan`: Multi-lens pre-mortem report that identifies upcoming failure modes before they become incidents.
- `Opus Escalation Gate`: Deterministic scoring check that blocks unnecessary high-cost model escalation.
- `Decision-Space Clamp`: Upfront reduction of candidate execution paths to a small set (`<= 3`) before deep analysis begins.
- `Compaction Loop`: Repeated summarize-and-prune cycle that keeps only decision-relevant state in active context.
- `Scope Policy`: `.claude/scope.json` allow/deny map used by Edit/Write hooks to block out-of-scope file changes.
- `Command Center Map`: Machine-readable graph that exposes agent hierarchy, linked skills, and prompt sources for UI/dashboard rendering.
- `Source Dossier`: Research lane for ecosystems and references that are useful context but are not default runtime tools.
